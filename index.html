Aprendendo Programação com JavaScript

Este é um curso formal e extenso, indicado àqueles que desejam aprender as bases da programação. Se você tem curiosidade sobre o que é programação, ou já sabe programar e apenas quer aprender JavaScript, este curso não é indicado para você. Nós abordaremos todos os conhecimentos necessários para se tornar um programador em qualquer linguagem, com foco em algoritmo. A linguagem JavaScript foi escolhida para o ensino apenas por sua ubiquidade e simplicidade. Para usar este curso, basta um browser moderno; não há configurações de ambiente necessárias.

Qualquer código utilizado ao longo dos exercícios pode ser executado diretamente no console do próprio browser. Para abrí-lo no Google Chrome, basta apertar F12 no Windows e Linux, ou alt+command+i no Mac, e clicar em Console. Caso utilize outro browser, ver esta lista de atalhos.

Este curso funciona offline também, para isso basta salvar a página e acessá-la de seu browser normalmente.
